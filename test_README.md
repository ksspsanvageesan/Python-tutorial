## 🚀 Getting Started

Follow these steps to run the project locally.

### 1️⃣ Clone the Repository

```
git clone https://github.com/your-username/project-name.git
cd project-name
```

### 2️⃣ Install Dependencies

```
pip install -r requirements.txt
```

### 3️⃣ Run the Project

```
python main.py
```
